# Amazon-Price-Tracker
This is  a Simple Amazon price Tracker Build using Flask and Python Automation libraries

## First run these commands in terminal in vscode
```python
python -m venv venv # this will make the virtual environment named--venv
venv\Scripts\Activate # this will activate the venv -- command is for windows only
```

## Install Dependencies in requirements.txt
```python pip
pip install -r requirements.txt
```

## you are ready to go 
```python
python app.py
```
